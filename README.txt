# ETL Projects

Este repositório contém diversos scripts de ETL organizados em três subcategorias principais: ETL Comercial, ETL PBI e ETL Prospecção e Vendas. Cada diretório contém os scripts específicos para a sua respectiva categoria.

## Estrutura do Repositório

- **ETL-Comercial**:
  - `etl_comercial.py`: Script para processos de ETL relacionados a dados comerciais.

- **ETL-PBI**:
  - `etl_pbi.py`: Script para processos ETL usados em dados de projeto que alimentam um dashboard no Power BI.

- **ETL-Prospecção-e-Vendas**:
  - `etl_prospeccao1.py`: Script de ETL para planilha de contato do setor comercial.
  - `etl_prospeccao2.py`: Script de ETL que gera uma planilha de empresas, na qual alimenta um dashboard.
  - `etl_prospeccao3.py`: Script para gerar uma planilha de contatos, na qual alimenta um dashboard.
